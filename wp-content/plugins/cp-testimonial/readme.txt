== Installation ==

There are 3 ways to install this plugin:

= 1. The Easy Method =

1. In your Admin, go to menu Plugins > Add
2. Search for CP Testimonials
3. Click to install
4. Activate the plugin
5. A new option under Settings menu named `CP Testimonials` will appear in your Admin

= 2. Install Through Zip File =

1. Download the plugin (.zip file) on the right column of this page
2. In your Admin, go to menu Plugins > Add
3. Select the tab "Upload"
4. Upload the .zip file you just downloaded
5. Activate the plugin
6. A new option under Settings menu named `CP Testimonials` will appear in your Admin

= 3. Through FTP =

1. Upload `CP Testimonials` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new option under Settings menu named `CP Testimonials` will appear in your Admin
